Welcome to the No Agenda Shownote Search! Just enter a search term and optionally select 1 or more topic filters. You can narrow down the topic list by just typing in the text field. If you want to see all notes for a set of topics leave the text search field blank and select the topics you want to display.

Disclaimer: Older episodes pre episode 490 have not yet been parsed and likely won't be any time soon due to a complete lack of free time.

If you are interested in helping out with parsing out some of the older shownotes or contributing any feature enhancements you'd like to see in the Shownote Search then hit me up on GitHub and I'll be happy to point you in the right direction.
